>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>Description<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<
TheThief est un jeux d'infiltration . Vous incarnez un joueur devant avancer de salle en salle pour aller dans la salle des coffres , récuperer le trésor et revenir à votre point 
de départ. Mais attention , vous faites du bruit et les gardes sont partout !!!
Pour vous aidez, vous posséderez une montre vous indiquant le temps qu'il vous reste,
Une map vous permettant de savoir où sont les gardes ainsi que votre position à chaque tour.
Vous pourrez rencontrer diverse objet dans les salles tel que :
la box qui vous permet de vous cacher pendant 3 tours (annulant le bruit réaliser)
le bait qui vous permet de leurer le garde (poser cette objet consommera sont énergie et fera du bruit dans les salles alentours)
Pour avancer, il vous faudre récuperer des clées à travers tout le niveau.
